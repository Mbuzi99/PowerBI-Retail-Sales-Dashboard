# Retail Store – Sales Analysis Dashboard 

This Power BI project provides insights into a retail store’s sales performance across South African cities. The dashboard highlights key business metrics such as total units sold, regional performance, and product category contribution.

---

Key Insights
- 45K+ units sold over time
- Johannesburg, KZN & East London lead in performance
- Top products contributing highest quantity
- Year-on-year sales trends analyzed with bar charts
- Interactive city-based filtering

---

Visuals Included
- Donut chart → Quantity Sold by City
- Pie chart → Sales Distribution by City
- KPI Card → Total Units Sold (45K)
- Bar chart → Sales by City and Year
- City Filters (Slicers)

---

Tools & Skills Used
| Feature | Description |
|--------|-------------|
| Power BI Desktop | Main reporting tool |
| Power Query | Data transformation |
| DAX | KPI measures & business logic |
| Data Modeling | Designed relationships for accuracy

 How to View the Report
